# Roadmap

## Versão 1.0 — Núcleo operacional

- Autenticação;
- Controle de usuários;
- Integração com Active Directory;
- Inventário básico;
- Agente estável;
- Helpdesk básico;
- Base de conhecimento inicial;
- Dashboard inicial.

## Versão 1.5 — Gestão de ativos

- Histórico completo de ativos;
- Garantias;
- Contratos;
- Licenciamento;
- Relatórios de inventário;
- Exportação CSV/PDF;
- Auditoria básica.

## Versão 2.0 — Atendimento avançado

- SLA;
- Escalonamento;
- Workflow de chamados;
- Aprovação;
- Notificações por e-mail;
- Categorização avançada;
- Pesquisa global.

## Versão 2.5 — Acesso remoto

- Integração com Apache Guacamole;
- Integração com RustDesk;
- Registro de sessões;
- Controle de permissão para acesso remoto;
- Histórico de acesso por ativo/chamado.

## Versão 3.0 — CMDB e governança

- CMDB;
- Relacionamento entre ativos;
- Serviços de TI;
- Gestão de mudanças;
- Gestão de problemas;
- Indicadores estratégicos.

## Versão 4.0 — Automação e IA

- Assistente interno;
- Sugestão automática de categoria;
- Resumo de chamados;
- Recomendação de artigos da base de conhecimento;
- Detecção de padrões recorrentes;
- Chatbot de atendimento.
