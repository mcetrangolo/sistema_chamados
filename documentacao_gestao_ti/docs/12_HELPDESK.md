# Módulo Helpdesk

## Objetivo

Gerenciar solicitações, incidentes e atendimentos técnicos da equipe de TI.

## Entidades principais

- Chamado;
- Categoria;
- Prioridade;
- Status;
- Técnico;
- Solicitante;
- Comentário;
- Anexo;
- SLA;
- Histórico.

## Status sugeridos

- Aberto;
- Em atendimento;
- Aguardando usuário;
- Aguardando fornecedor;
- Resolvido;
- Encerrado;
- Cancelado.

## Prioridades

- Baixa;
- Média;
- Alta;
- Crítica.

## Funcionalidades essenciais

- Abrir chamado;
- Atribuir técnico;
- Alterar status;
- Registrar comentários;
- Relacionar ativo;
- Anexar arquivos;
- Registrar solução;
- Encerrar chamado;
- Reabrir chamado;
- Pesquisar chamados.

## Funcionalidades futuras

- SLA;
- Escalonamento;
- Aprovação;
- Workflow;
- Pesquisa inteligente;
- Sugestão de solução baseada na base de conhecimento;
- Relatórios de atendimento.
