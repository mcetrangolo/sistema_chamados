# Plataforma Integrada de Gestão de TI

## Objetivo

Desenvolver uma plataforma integrada de Gestão de Tecnologia da Informação, voltada inicialmente para uso interno em órgão público municipal/autarquia, com possibilidade futura de evolução para produto reutilizável em outras organizações.

A plataforma deve centralizar:

- Inventário de equipamentos;
- Helpdesk;
- Base de conhecimento;
- Dashboard;
- Acesso remoto;
- Gestão de contratos;
- Gestão de licenças;
- Gestão de garantias;
- Documentação técnica;
- Monitoramento;
- Auditoria;
- Relatórios gerenciais.

## Escopo principal

A solução deve resolver problemas práticos de pequenas e médias equipes de TI, especialmente em ambientes públicos, onde é comum haver falta de integração entre chamados, inventário, documentação, contratos e controle patrimonial.

## Não é objetivo

- Reproduzir integralmente o GLPI;
- Reproduzir integralmente o iTop;
- Reproduzir integralmente o Znuny;
- Criar uma ferramenta excessivamente complexa;
- Implementar ITIL completo antes de consolidar o núcleo operacional;
- Depender de plugins externos para funcionalidades essenciais.

## Princípio central

A plataforma deve ser simples de operar, modular, integrada e orientada a APIs.
