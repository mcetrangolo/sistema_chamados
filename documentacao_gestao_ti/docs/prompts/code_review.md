# Prompt — Code Review

Revise o código alterado considerando:

- Segurança;
- Duplicação;
- Clareza;
- Performance;
- Padrões do projeto;
- Separação de responsabilidades;
- Possíveis bugs;
- Consultas ineficientes;
- Falhas de permissão;
- Falhas de validação.

Use como referência:

- `docs/03_PRINCIPIOS.md`
- `docs/17_SEGURANCA.md`
- `docs/18_PADROES_CODIGO.md`

Retorne:

- Problemas encontrados;
- Severidade;
- Sugestão de correção;
- Arquivos impactados.
