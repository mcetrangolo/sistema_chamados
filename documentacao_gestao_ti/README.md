# Kit de Documentação — Plataforma de Gestão de TI

Este pacote contém uma estrutura inicial de documentação para orientar o desenvolvimento da plataforma de gestão de TI.

## Como usar

1. Extraia este conteúdo na raiz do repositório.
2. Peça ao VS Code/Codex/Copilot/Claude Code para ler `docs/23_CONTRIBUICAO_IA.md`.
3. Para novas funcionalidades, use os prompts em `docs/prompts/`.
