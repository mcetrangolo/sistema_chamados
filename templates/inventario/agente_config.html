Fluxo Windows

O instalador .exe e um pacote interno sem assinatura digital. Antivírus como Avast/CyberCapture podem enviar o arquivo para analise por ele ser novo e executar PowerShell para coletar inventario. Quando isso ocorrer, use o ZIP sem EXE ou assine digitalmente o instalador antes de distribuir em massa.
  1. Baixar o instalador pela tela de inventario.
  2. Executar como Administrador na estacao Windows.
  3. Informar o endereco do servidor, por exemplo 192.168.0.10:8000.
  4. Confirmar a primeira coleta e verificar o ativo na lista de inventario.

Se o Avast/CyberCapture bloquear o .exe, use Baixar Windows ZIP, extraia o pacote e execute InstalarAgente.cmd como Administrador. O codigo do agente fica visivel dentro do ZIP.

Fluxo Linux

  1. Baixar o arquivo sistema-chamados-agent-linux.sh.
  2. Copiar para a maquina Debian, Ubuntu, Proxmox ou VM Linux.
  3. Executar sudo sh sistema-chamados-agent-linux.sh.
  4. Informar o endereco do servidor e confirmar a primeira coleta.

O agente Linux usa systemd timer quando disponivel: executa no boot e a cada 6 horas. Se nao houver systemd, usa cron.

Como desinstalar

A nova instalacao cria a pasta Sistema Chamados Agent no Menu Iniciar com a opcao Desinstalar agente. Tambem deve aparecer em Programas e Recursos do Windows.

Se precisar remover manualmente, execute como Administrador:

powershell.exe -NoProfile -ExecutionPolicy Bypass -File C:\ProgramData\SistemaChamadosAgent\uninstall.ps1

No Linux, execute como root:

sudo sh /opt/sistema-chamados-agent/uninstall.sh
