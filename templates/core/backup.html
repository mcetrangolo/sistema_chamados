Restaurar backup

A restauração substitui a base atual e os arquivos de mídia. O sistema cria um backup prévio automaticamente antes de restaurar.

{% csrf_token %}
Use o arquivo `.zip` baixado anteriormente pela tela de backup.

Ambiente atual

Banco local SQLite

O backup cria um ZIP com a base `db.sqlite3` e a pasta `media`.

Cenário de catástrofe

Guarde o `.zip` em outro local. Se perder a máquina ou a pasta do sistema, instale novamente a aplicação e use “Selecionar arquivo do meu computador” para restaurar esse backup externo.

Atenção

  • Reinicie após restaurarDepois do restore, pare e suba o servidor novamente.
  • Apagar é definitivoBaixe uma cópia externa antes de excluir backups antigos.
  • Proteja arquivosBackups podem conter dados pessoais e anexos sensíveis.
